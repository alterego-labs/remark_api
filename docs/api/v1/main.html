  }

Users

/users/{login} get

get

Provides the information for user

get /users/{login}

Provides the information for user

  • Request
  • Response

URI Parameters

  • User login: required (string)

HTTP status code 200

When user with requested login found

Body

Type: application/json

Example:

{
  "data": {
    "user": {
      "login": "Some login"
    }
  }
}

HTTP status code 404

When no user with requested login exists

Body

Type: application/json

Example:

{
  "data": {
    "errors": [
      "User not found!"
    ]
  }
}
